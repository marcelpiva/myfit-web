# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-01-26

### Added
- Account deletion page (`/profile/delete`) for Google Play Store compliance
  - Multi-language support (Portuguese, English, Spanish)
  - Clear explanation of data deletion process
  - Two deletion methods: in-app and email
  - Deletion request form with confirmation
  - Warning about irreversible action
  - Contact information for support

## [0.1.0] - 2025-01-01

### Added
- Initial release
- Landing page with hero section
- Features showcase
- Pricing plans (Free, Pro, Enterprise)
- FAQ section
- Contact page
- About page
- Careers page
- Privacy policy page
- Terms of service page
- Login and registration pages (UI only)
- Multi-language support (PT, EN, ES)
- Dark mode support
- Responsive design
- Framer Motion animations
